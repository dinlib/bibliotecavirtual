# Biblioteca Virtual

Biblioteca Virtual do Departamento de Informática da UEM.  
Está biblioteca é aberta, portanto, do conhecimento de todos, inclusive dos professores. Devido a isso, encorajamos a usá-la com sabedoria e em hipótese alguma realizar cópia dos trabalhos contidos aqui. Use-os apenas como uma referência para o desenvolvimento da sua própria solução.

Se você deseja contribuir com a biblioteca, por favor, leia [aqui](CONTRIBUA.md).

Considere visitar a segunda parte da nossa biblioteca no [Google Drive](http://abre.ai/libdin).
